Effets de bord de la validation enseignant par liste de suivi

Si une "liste de suivi" (module non standard) est utilisée dans le cours pour marquer les accomplissments, certaines ativités peuvent être validées par les enseignants sans aucune interaction préalable de l'étudiant dans le cours (et donc sans traces).

Ceci est une situation possible et normale et le rapport affiche des informations "justes" dans ses différents indicateurs d'usage.
